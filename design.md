# Academic Portfolio Design Style

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a2332) - Professional, academic authority
- **Secondary**: Warm Gray (#6b7280) - Balanced, sophisticated
- **Accent**: Copper Gold (#b45309) - Achievement, excellence
- **Background**: Soft Cream (#fefefe) - Clean, readable
- **Text**: Charcoal (#374151) - High contrast, accessible

### Typography
- **Display Font**: "Tiempos Headline" - Bold serif for headings, academic gravitas
- **Body Font**: "Suisse Int'l" - Clean sans-serif for readability
- **Code Font**: "JetBrains Mono" - Technical content, research precision

### Visual Language
- **Academic Excellence**: Clean, structured layouts with generous white space
- **Research Focus**: Data visualizations, charts, and technical diagrams
- **Professional Credibility**: Consistent formatting, scholarly presentation
- **Innovation**: Modern web technologies showcasing technical skills

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth transitions and micro-interactions
- **ECharts.js**: Research data visualization and academic charts
- **p5.js**: Creative coding background effects and data art
- **Splitting.js**: Typography effects for headings
- **Splide.js**: Image carousels for research projects

### Effect Implementation
- **Hero Background**: Subtle particle system representing neural networks
- **Text Animation**: Staggered reveal for academic achievements
- **Data Visualization**: Interactive charts showing research impact
- **Scroll Motion**: Gentle parallax effects for depth
- **Hover Effects**: 3D tilt on project cards with shadow expansion

### Styling Approach
- **Grid System**: Structured layout reflecting academic organization
- **Consistent Spacing**: 8px base unit for harmonious proportions
- **Subtle Shadows**: Soft depth without overwhelming content
- **Responsive Design**: Mobile-first approach for accessibility
- **Performance**: Optimized loading for academic networks

### Header Effect
- **Animated Background**: Flowing data streams representing ML algorithms
- **Gradient Overlay**: Subtle color transitions maintaining readability
- **Typography Animation**: Letter-by-letter reveal of name and title
- **Interactive Elements**: Hover states on navigation and social links

This design creates a sophisticated academic portfolio that demonstrates both technical competence and scholarly professionalism, ideal for graduate school applications and research-focused career opportunities.
